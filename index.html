About My Topic

SimpleOcean is an enviromental website that provides information about the current state of ocean and the environmental harms that it currently faces. The main purpose of the website is to be a fun, but also an informative guide on damaged ocean-realted topics such as corals, beaches, and ocean ecosystems. It will contain information on how to help the ocean and the environment, as well as tips on how to be more sustainable.
